# 快速部署到 GitHub Gist

## 步骤（30秒）

1. 打开 https://gist.github.com
2. 登录你的 GitHub 账号
3. 粘贴 `index.html` 的全部内容
4. 文件名填写：`index.html`
5. 点击 **Create secret gist** 或 **Create public gist**
6. 复制 gist 的 URL（例如 `https://gist.github.com/你的用户名/abc123`）
7. 发给我

## 获取直链

我会帮你转换成可直接访问的链接：
- Gist链接：`https://gist.github.com/用户名/abc123`
- 直链：`https://gistcdn.githack.com/用户名/abc123/raw/index.html`

这个链接永久有效，无需服务器！
